Cake Manager Micro Service (fictitious)
=======================================
The current application is modified to use spring boot libraries.

To build the code run 'mvn clean install'

After that, To run the server locally execute the following command:

`java -jar target\cake-manager-1.0-SNAPSHOT.war`

and access the following URL:

`http://localhost:8282/`


TO access RestAPI use following URL: 
'http://localhost:8282/cakes'

No additional features provided. Only functionality added.
